HSL Developer Community is open for all individuals and organisations interested in the Open Data and Open Source software regarding Helsinki Region Transport HSL or wider. HSL is the public authority that plans and procures the public transport (ferry, tram, metro, train, bus) within its member municipalities (Helsinki, Vantaa, Espoo, Kauniainen, Kirkkonummi, Kerava, and Sipoo).

Journey planning


Digitransit (project page)

HSL's next-generation journey planner, in cooperation with Finnish Transport Agency's national Journey.fi.

Digitransit project includes digitransit-ui user interface and back-end services providing APIs for public transportation data, geocoding and maps.

  • Developer page
  • GraphQL console, including schema documentation
  • Data container, including GTFS feed
  • Digitransit application - built on HTML5, React, Relay, GraphQL, MQTT (source code)

Live maps

Most vehicles in the HSL area publish their status to a MQTT server. MQTT clients can subscribe to these status messages and filter them based on e.g. transport mode and geographical area.

  • MQTT API documentation
  • MQTT demo - displays vehicles on a map (source code)
